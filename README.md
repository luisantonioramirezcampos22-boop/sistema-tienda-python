# Sistema Integral de Gestión de Tienda
Este sistema controla el catálogo de productos y gestiona las ventas aplicando estructuras de datos (listas y diccionarios) junto con algoritmos de búsqueda y ordenamiento manual en Python.

## Estructura del Proyecto
- `producto.py`: Es la estructura y atributos de la entidad Producto.
- `inventario.py`: Es la lógica de negocio, control de stock, alertas y algoritmos.
- `main.py`: Es la interfaz de consola interactiva para el usuario.

## Cómo Ejecutar el Programa
Debes asegurar de tener Python instalado y ejecuta el siguiente comando en la terminal: python main.py
